能让你在云商店上面的wordpress访问速度更快

== Description ==

能让你在云商店上面的wordpress访问速度更快， 如果你在云商店上面安装了wordpress，并且使用了没有备案的域名， 此时为保证网站正常访问，是通过海外的代理来访问网站，必然速度比较慢， 云商店加速插件， 能让资源文件不走代理， 直接通过二级域名访问，从而提高了网站的访问速度

== Installation ==

1. Upload the `ysd-comment` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the settings using "WP Connect" under Settings panel.

下载 云商店打包插件，上传ysd-comment目录及其文件到 "/wp-content/plugins/" 插件目录，在后台管理中激活插件，到设置页面开启功能并设置等.
